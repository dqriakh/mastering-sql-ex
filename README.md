## 📌 О проекте

В этом репозитории собраны мои решения практических упражнений с сайта **[sql-ex.ru](https://sql-ex.ru/)**.
Сайт предлагает отличные задачи для отработки навыков SQL — от простых запросов до сложных многоуровневых конструкций.

**База данных:** "Компьютерная фирма" (таблицы Product, PC, Laptop, Printer)

---

## 🎯 Цели проекта

- 📊 Систематизировать знания SQL
- 🏋️ Отработать различные типы запросов
- 📂 Создать портфолио для демонстрации навыков

---

## 📋 Схема базы данных

```sql
Product(maker, model, type)
PC(code, model, speed, ram, hd, cd, price)
Laptop(code, model, speed, ram, hd, price, screen)
Printer(code, model, color, type, price)

Product — производители, модели и типы продуктов

PC — персональные компьютеры

Laptop — портативные компьютеры (ноутбуки)

Printer — принтеры
